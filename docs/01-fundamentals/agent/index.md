# AI Agent
자율적으로 작업을 수행하는 AI Agent 설계와 구현을 정리합니다.
## 다루는 내용
- **Agent 개념** — ReAct, Plan-and-Execute 패턴
- **LangGraph** — 상태 기반 워크플로우, 멀티에이전트
- **Tool Use** — 외부 API 연동, 함수 호출
- **멀티에이전트** — 에이전트 간 협업 구조
## 주요 도구
| 도구 | 용도 |
|------|------|
| LangGraph | 에이전트 워크플로우 |
| LangChain | 도구 & 체인 |
