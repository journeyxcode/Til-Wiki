# 딥러닝
딥러닝 아키텍처와 학습 방법론을 정리합니다.
## 다루는 내용
- **신경망 기초** — 퍼셉트론, 활성화 함수, 역전파
- **CNN** — 이미지 분류, 객체 탐지
- **RNN / LSTM** — 시퀀스 모델링
- **Transformer** — Attention 메커니즘, BERT, GPT
