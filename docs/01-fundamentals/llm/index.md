# LLM (대규모 언어 모델)
LLM의 개념, 활용, 프롬프트 엔지니어링을 정리합니다.
## 다루는 내용
- **LLM 개념** — GPT, Claude, Gemini 아키텍처 비교
- **프롬프트 엔지니어링** — Zero-shot, Few-shot, CoT
- **파인튜닝** — LoRA, QLoRA, RLHF
- **LangChain** — 체인 구성, 메모리, 툴 연동
## 주요 도구
| 도구 | 용도 |
|------|------|
| LangChain | LLM 프레임워크 |
| OpenAI API | GPT 모델 |
| Groq | 빠른 추론 |
| Google AI Studio | Gemini 모델 |
