# Docusaurus

Docusaurus는 Meta(Facebook)가 만든 오픈소스 정적 사이트 생성기입니다.
Markdown 기반으로 문서를 작성하고 React로 커스터마이징할 수 있어요.

## 왜 Docusaurus인가?

| 특징 | 내용 |
|------|------|
| 빠른 시작 | 명령어 하나로 프로젝트 생성 |
| Markdown 지원 | MDX(Markdown + JSX) 지원 |
| 자동 사이드바 | 폴더 구조 기반 자동 생성 |
| 다크모드 | 기본 제공 |
| 검색 | Algolia DocSearch 연동 |
| 무료 배포 | GitHub Pages 연동 |

## 이 문서에서 다루는 내용

- **설치 & 시작** — 로컬 환경 설정부터 첫 실행까지
- **GitHub Pages 배포** — GitHub Actions 자동 배포 설정
- **테마 커스터마이징** — 색상, 폰트, 레이아웃 변경
- **폴더 구조** — 사이드바 구성 및 카테고리 정렬
- **삽질 & 팁** — 자주 만나는 오류와 해결 방법
