폴더 구조 & 사이드바

기본 폴더 구조​

docs/
├── 01-category/
│   ├── _category_.json   ← 카테고리 설정
│   ├── index.md          ← 카테고리 클릭 시 보이는 페이지
│   └── 01-post.md        ← 문서
└── intro.md

category.json​

폴더의 사이드바 표시 이름과 순서를 설정해요:

{
  "label": "AI 기본기",
  "position": 1,
  "collapsible": true,
  "collapsed": true
}
옵션설명
label사이드바에 표시될 이름
position순서 (숫자가 작을수록 위)
collapsible접기 가능 여부
collapsed기본 접힘 여부

문서 순서 정렬​

방법 1 — 파일명에 숫자 붙이기​

01-math-stats.md
02-ml.md
03-dl.md

방법 2 — frontmatter로 지정 (추천 ⭐)​

---
title: 머신러닝
sidebar_position: 2
---

index.md 활용​

카테고리 폴더를 클릭했을 때 보여줄 페이지예요. sidebar_position: 1 로 설정하면 항상 맨 위에 위치해요:

---
title: 개요
sidebar_position: 1
---

# AI 기본기
이 섹션에서 다루는 내용 소개...

sidebars.js 설정​

특정 순서로 사이드바를 구성하고 싶을 때 사용해요:

const sidebars = {
  tutorialSidebar: [
    'intro',
    {
      type: 'category',
      label: 'AI 기본기',
      items: [{ type: 'autogenerated', dirName: '01-fundamentals' }],
    },
    {
      type: 'category',
      label: '프로젝트',
      items: [{ type: 'autogenerated', dirName: '02-projects' }],
    },
  ],
};

빈 폴더 유지하기​

Git은 빈 폴더를 추적하지 않아요. .gitkeep 파일을 넣으면 빈 폴더도 GitHub에 올라가요:

touch docs/01-category/.gitkeep
