# FastAPI
Python 기반 고성능 API 프레임워크 FastAPI를 정리합니다.
## 다루는 내용
- **기본 구조** — 라우터, 의존성 주입, 미들웨어
- **데이터 검증** — Pydantic 모델
- **인증** — JWT, OAuth2
- **배포** — Docker, Uvicorn
## 기술 스택
| 기술 | 용도 |
|------|------|
| FastAPI | API 프레임워크 |
| Pydantic | 데이터 검증 |
| SQLAlchemy | ORM |
| Docker | 컨테이너화 |
