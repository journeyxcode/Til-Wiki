## 📖 소개

AI LOG는 [Docusaurus](https://docusaurus.io/) 기반의 기술 문서 사이트입니다.

Android 개발 경력을 바탕으로 AI 엔지니어로 전환하며 학습한 내용,
프로젝트 경험, AI 최신 소식을 체계적으로 정리합니다.
